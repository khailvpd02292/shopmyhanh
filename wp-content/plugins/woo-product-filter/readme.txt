== Description ==
[WooCommerce Product Filter plugin](https://woobewoo.com/product/woocommerce-filter/ "WooCommerce Product Filter plugin") allows customers filter your products/items by price, categories, tags, taxonomy and attributes. Each filter criteria customizable, giving you maximum control.
This is a perfect way to increasing sales make the shopping process more enjoyable.

= WooCommerce Product Filter Main Features =

**AJAX live search**
Make filtering even more comfortable by enabling AJAX search or let the process start as soon as user selects all the options and presses the filtering button.

**Variety of options for sorting products on the page**
Product search plugin for WooCommerce allows sorting your products by popularity, rating, price, newness, alphabetical ascender or descender.

**Create automatic price range settings with fixed step values**
Set them manually and specify the exact value of every step as well.

**Add filter by categories, tags and attributes**
Visitors can find exactly what they needs with the maximum probability.

**Variations of display at the front end**
  Display filters in any convenient way: drop-down, multiple dropdown, checkbox list and the form of icons for attribute filters.

**Create new product filters in a few clicks**
Attach them easily to every page using widget or shortcodes.

== Installation ==

a) VIA FTP
Download zip archive with WooCommerce Product Filter by WooBeWoo plugin
Extract files
Copy the folder with plugin
Open via ftp folder \wp-content\plugins\
Paste plugin folder into here
Go to admin panel => Plugins => WooCommerce Product Filter by WooBeWoo press activate

B) DIRECT UPLOAD VIA ADMIN AREA
Download zip archive with plugin
Go to admin panel => Plugins => Add new
Choose the archive with plugin and upload it
Wait until you get the notification installation went successfully
Press Activate button

C) VIA WORDPRESS REPOSITORY
Navigate to admin panel => Plugins => Add new
Type \"WooCommerce Filter Plugin by Woobewoo"\ in the search field
Choose our plugin from the list and click \"Install\"
Wait until you get the notification installation went successfully
Press Activate button

== Frequently Asked Questions ==

= How can I add product filter to my website? =
To manage with an installation and plugin settings, check tutorial [Main Settings](https://woobewoo.com/documentation/woocommerce-filter-main-settings/ "Product Filter plugin main settings") and [Getting Started with WooCommerce product filter](https://woobewoo.com/documentation/getting-started-with-woocommerce-filte "Getting Started with WooCommerce product filter WordPress plugin")
= What are the criteria I can sort by =
Now there is an ability to sort results by popularity, rating, newness, price low to high and vice versa, randomly, by name, or by default order.
= Is the Product filter plugin responsive? 
WooCommerce Product Filter is fully compatible and responsive! Plugin is working and displaying perfect on tablet or phone, making it easy and convenient for customers to enjoy their shopping time and make purchases on all devices.  
= My WooCommerce Product Filter plugin is not working? =
If you have problems with the work of plugin or something is not working as it should, follow this steps:
 * Test the plugin with a WordPress default theme, to be sure that the error is not caused by the theme you are currently using.
 * Deactivate all plugins you are using and check if the problem is still occurring.
 * Ensure that plugin version, theme version and WordPress are updated and the problem has not already been solved in a later plugin update.
If none of the this advices help you, contact us via [support form on our website](https://woobewoo.com/contact-us)and describe your problem.

= WooCommerce Product Filter plugin is not available in my language. What shall I do? =
You may help us translate the plugin into your language and get the PRO version of plugin for free as soon as it is released. If this method seems to be too complicated, please send us a request and we will let you know as soon as WooBeWoo Filter plugin will be available in your localization.
